# vegetables_web

## Installation

Use the package manager [npm](https://www.npmjs.com/get-npm) to install APIs.

```bash
npm install
```

## Usage

### For development environment:

```bash
npm run start-dev
```

##### OR:

```bash
npm run start
```

### For deployment environment:

##### (Linux OS & MacOS):

```bash
npm run start-prod
```

##### (Windows OS):

```bash
npm run start-prod-windows
```
